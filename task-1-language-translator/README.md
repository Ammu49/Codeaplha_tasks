### AI Language Translation Tool
### Overview

The AI Language Translation Tool is a web application that allows users to translate text between multiple languages using a translation API. Users can enter text, select source and target languages, and instantly receive translated output.

This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

### Features

Translate text between multiple languages
Select source and target languages
User-friendly interface
Fast and accurate translations using API integration
Copy translated text to clipboard
Responsive design

Clone the Repository

git clone

cd language-translator

Install Dependencies

### Frontend:

npm install

### Backend:

npm install

### Configure Environment Variables

Create a .env file in the server directory and add:

AZURE_TRANSLATOR_KEY=your_api_key

AZURE_TRANSLATOR_REGION=your_region

PORT=3001

Run the Application

### Backend:

npm run server

### Frontend:

npm run dev

### Usage

Enter text in the input box.
Select the source language.
Select the target language.
Click the Translate button.
View the translated output.
Copy the translated text if needed
